## Requirements
[Docker](https://www.docker.com/)

## Setup Environment
Build Docker image:

```bash
docker build -t pancakes-sql-training .
```

Run Docker image `pancakes-sql-training`:

```bash
docker run -p 3306:3306 pancakes-sql-training:latest
```
